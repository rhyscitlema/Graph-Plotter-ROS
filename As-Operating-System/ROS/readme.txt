
**********************************
On Ubuntu Linux OS
**********************************

- Go to the command line and change directory to that containing this Readme.txt file.

- Make sure that the kernel was already compiled and is found in the folder "files" .
- IMPORTANT: Kernel file name + file extension must not exceed 11 characters.
- NOTE: Kernel file name is considered to be written in CAPITAL LETTERS.

- Make sure that the program file "nasm" found inside the folder "bootloader" is marked as executable.

- Make sure that the file "build.sh" is marked as executable.

- Finally, execute: ./build.sh
